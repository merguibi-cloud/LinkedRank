# Guide des Agents IA - LinkedAgents

## 🤖 Qu'est-ce qu'un Agent IA ?

Un **Agent IA** est un assistant automatisé qui travaille pour vous sur LinkedIn. Chaque agent a une spécialité et peut :
- Générer du contenu automatiquement
- Analyser les tendances
- Suggérer des réponses aux commentaires
- Recommander des stratégies de croissance

## 📋 Les 5 Agents Disponibles

### 1. ✍️ Content Creator
**Rôle** : Génère du contenu personnalisé basé sur votre style et vos objectifs.
- Analyse votre historique de posts
- Crée des posts adaptés à votre audience
- Respecte votre ton et votre style

### 2. 🔍 Trend Hunter
**Rôle** : Détecte les tendances et vous alerte sur les opportunités de contenu.
- Surveille les hashtags populaires
- Identifie les sujets viraux
- Suggère des angles d'actualité

### 3. 💬 Engagement Manager
**Rôle** : Analyse les commentaires et suggère des réponses personnalisées.
- Lit les commentaires sur vos posts
- Propose des réponses adaptées
- Maintient l'engagement avec votre communauté

### 4. 📈 Growth Strategist
**Rôle** : Analyse vos performances et recommande des stratégies de croissance.
- Étudie vos métriques
- Identifie ce qui fonctionne
- Propose des améliorations

### 5. 🤝 Network Builder
**Rôle** : Identifie les connexions stratégiques et optimise votre réseau.
- Trouve des profils pertinents
- Suggère des personnes à suivre
- Optimise votre réseau professionnel

## ⚙️ Comment Utiliser un Agent

### Étape 1 : Activer l'Agent
1. Allez sur la page **Agents IA**
2. Cliquez sur le **switch** à côté de l'agent souhaité
3. L'agent passe de "En pause" à "Actif"

### Étape 2 : Configurer l'Agent
1. Cliquez sur **Config** pour personnaliser l'agent
2. Définissez :
   - Les sujets/thèmes à traiter
   - Le ton souhaité (professionnel, décontracté, etc.)
   - La fréquence de génération

### Étape 3 : Planifier l'Agent
1. Cliquez sur **Planifier** pour définir le calendrier
2. Choisissez :
   - Les jours de la semaine (Lun, Mar, Mer, etc.)
   - Les heures de génération (8h, 12h, 18h, etc.)
   - Le nombre de tâches par jour

### Étape 4 : Approuver ou Rejeter
1. Allez dans l'onglet **Approbations**
2. Consultez les tâches générées par les agents
3. **Approuvez** pour publier ou **Rejetez** pour ignorer

## 🔄 Les 3 Modes d'Autonomie

### Mode Supervisé (par défaut)
- L'agent génère du contenu
- **Vous devez approuver** chaque tâche avant publication
- Contrôle total sur ce qui est publié

### Mode Semi-Autonome
- L'agent génère du contenu
- Les tâches simples sont auto-approuvées
- Les tâches complexes nécessitent votre approbation

### Mode Autonome
- L'agent génère ET publie automatiquement
- Aucune approbation requise
- Idéal pour les agents de confiance après test

---

## 📅 Planification Automatique

### Comment ça marche ?
1. L'agent vérifie le calendrier toutes les minutes
2. Si c'est l'heure planifiée → génère une tâche
3. Selon le mode d'autonomie :
   - **Supervisé** : La tâche attend votre approbation
   - **Autonome** : La tâche est publiée automatiquement

### Exemple de Configuration
- **Jours** : Lundi, Mercredi, Vendredi
- **Heures** : 9h, 14h
- **Tâches/jour** : 1
- → L'agent génère 1 post à 9h et 1 à 14h, 3 fois par semaine

## 🧪 Tester un Agent

### Test Manuel
1. Activez l'agent (switch ON)
2. Cliquez sur le bouton **⚡** (éclair) pour forcer une génération
3. Allez dans **Approbations** pour voir le résultat
4. Approuvez ou rejetez selon la qualité

### Vérifier les Logs
1. Allez dans l'onglet **Activité**
2. Consultez l'historique des actions de chaque agent
3. Identifiez les erreurs éventuelles

## ❓ FAQ

### Q: Pourquoi mon agent ne génère rien ?
**R**: Vérifiez que :
- L'agent est activé (switch ON)
- La planification est configurée
- L'heure actuelle correspond au calendrier

### Q: Comment publier sur LinkedIn ?
**R**: 
1. Approuvez une tâche dans l'onglet Approbations
2. Le contenu est automatiquement publié si LinkedIn est connecté
3. Ou copiez le contenu et publiez manuellement

### Q: Puis-je modifier le contenu avant publication ?
**R**: Oui ! Dans l'onglet Approbations, vous pouvez éditer le contenu avant d'approuver.

### Q: Comment arrêter un agent ?
**R**: Cliquez simplement sur le switch pour le désactiver. Il passera en "En pause".

## 🚀 Bonnes Pratiques

1. **Commencez en mode Supervisé** - Testez la qualité avant d'automatiser
2. **Configurez bien vos sujets** - Plus c'est précis, meilleur est le contenu
3. **Vérifiez régulièrement** - Même en mode autonome, consultez les publications
4. **Ajustez la fréquence** - Trop de posts peut nuire à l'engagement
5. **Utilisez les analytics** - Mesurez ce qui fonctionne et ajustez
